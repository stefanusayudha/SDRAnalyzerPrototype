# FFT Analyzer Symulator

options:
  -h Show help
  -f Spectrum list / list of frequencies seprated by ","
  -s Sampling rate. Default = 44100
  -d Duration in second. Default = .5

example:
./fft_analyzer.sh -f 100,200 -d .1 -s 44100
